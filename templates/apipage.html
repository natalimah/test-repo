API Guide

Preview

This guide will help you to understand the api and how to use it.
It will help you to make automation and script base on the api.

Get

When you want to list exsiting data you have two options.
I. you can list all the data by curl the "/api/products" route.

curl $server_ip/api/products

II. On the other hand you can get data of spesific product by curl same route with the product name.

curl $server_ip/api/products/$product_name


you can also use your browser to reach the route
